Install as an app

Add The Flavour Editors to your home screen so it opens like a proper app — no browser chrome, full screen.

iPhone / iPad

Open in Safari → tap the Share button → "Add to Home Screen" → Add.

Android

Open in Chrome → tap the three-dot menu → "Add to Home screen" → Add.

Desktop (Chrome / Edge)

Look for the install icon in the address bar, or go to browser menu → "Install The Flavour Editors".

Danger zone

Clear all data

Permanently deletes all recipes, saved plans and settings. Export a backup first.
